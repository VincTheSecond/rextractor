Learn more about RExtractor

RExtractor is a system that extracts a knowledge base from raw unstructured texts. The base is designed as a set of entities and their relations and is represented in an ontological framework.

The extraction pipeline processes input texts by linguistically-aware tools and extracts entities and relations from their syntactic representation. Consequently, the extracted data is represented according to the Linked Data principles. The system is designed both domain and language independent and provides users with data for more intelligent search than full-text search.

RExtractor consists of four compoments:

  1. Conversion -- a largely technical component converting various input formats into an internal representation.

  2. Natural Language Processing -- a linguistic component providing various analyses of input texts, namely sentence segmentation, tokenization, morphological analysis, part-of-speech tagging, and syntactic parsing.

    Currently employed procedures come from Treex, a highly modular NLP software system. The NLP component outputs a dependency tree for each sentence in the input texts.
