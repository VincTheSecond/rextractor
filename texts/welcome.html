Welcome to RExtactor Web Interface

RExtractor is a system that extracts a knowledge base from raw unstructured texts. The base is designed as a set of entities and their relations and is represented in an ontological framework. This RExtraction demo is ready for processing Czech legal documents (e.g. laws and court decisions).

To play with RExtractor, you can submit a new document into the joblist or browse already processed documents.
