Welcome to RExtactor Web Interface

RExtractor is a system for extracting a knowledge base from unstructured texts. The knowledge base is a set of entities and relations between them. This RExtraction DEMO installation is ready for processing Czech legal documents (e.g. laws and court decisions).

To play with RExtractor, you can submit new document into the joblist or browse already processed documents.
